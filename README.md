## 📜 Installation

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/yourusername/youruserbot.git

2. Install Dependencies

pip install -r requirements.txt


3. Configuration
Edit the config.py file to fill in your bot token and settings.


4. Run the Bot
Start with:

python main.py
